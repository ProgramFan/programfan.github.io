About the theme

This site forks from jekyllrb.com and modifyed to enable latex, fontawesome and asciidoctor integration. All posts are written in asciidoc and converted to html by jelyll-asciidoc. Feel free to clone this site from its git repository.

Tests

LaTeX support through KaTeX

LaTeX is supported well through KaTeX. For example, $ a^2 + b^2 = c^2 $ is rendered as $a^2 + b^2 = c^2$, and \[ a^2 + b^2 = c^2 \] is rendered as \[ a^2 + b^2 = c^2 \].
